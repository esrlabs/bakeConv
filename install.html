How to install bake converter

bakeConverter is a ruby gem. It runs with ruby 1.9 and above.

bakeConverter and its depedencies can be found on rubygems.
Installing bake is very easy: 

gem install bakeConv

This is how the installation could look like: 

>>> gem install bakeConv
Successfully installed bakeConv-1.1.1
